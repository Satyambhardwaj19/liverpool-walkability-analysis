# Outputs

This folder contains visual outputs from the Liverpool Walkability Analysis project.

## Files

### `walking_network_map.png`

Static map showing the structure of Liverpool's walking network.

### `static_magnitude_map.png`

Static map showing street connectivity using node degree values.

### `interactive_walkability_preview.png`

Screenshot preview of the interactive Folium map. The full interactive map is generated inside the notebook.

## Purpose

These images are used in the main README so that viewers can understand the project outputs without opening the notebook first.
