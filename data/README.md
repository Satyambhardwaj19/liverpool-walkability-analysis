# Data

This project uses street network data retrieved from OpenStreetMap through OSMnx.

The data is not stored permanently in this repository because the notebook retrieves the walking network directly from OpenStreetMap. This keeps the repository lightweight and makes the workflow easier to reproduce.

## Dataset Source

- Source: OpenStreetMap
- Accessed through: OSMnx
- Location: Liverpool, United Kingdom
- Network type: Walking network

## Main Data Used

- Network nodes
- Network edges
- Street connectivity
- Node degree values
- Geographic geometry for mapping

## Note

The notebook contains the full workflow for retrieving, preparing, analysing, and visualising the walking network data.
