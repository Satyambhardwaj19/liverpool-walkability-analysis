# Notebooks

This folder contains the main analysis notebook for the Liverpool Walkability Analysis project.

## Notebook

### `liverpool_walkability_analysis.ipynb`

This notebook includes the full workflow for:

- retrieving Liverpool walking network data from OpenStreetMap using OSMnx
- converting the network into GeoDataFrames
- analysing street connectivity using node degree
- creating static walking network maps
- creating an interactive walkability map
- summarising key spatial patterns

## Note

The notebook is written as a portfolio version of the project, with cleaner structure, explanations, and outputs for GitHub.
